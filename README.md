# Javascript_Calculator

The final project of the Odin Project: Fundamentals.  This is a javascript calculator that is able to add, subtract, multiply, and divide two numbers.

The buttons will include:
    1. Buttons 0-9
    2. An = button
    3. A multiply operator
    4. A divide operator
    5. An add operator
    6. A subtract operator
    7. A clear button

To use simply enter any number, followed by an operator, then another number followed by the equals button.  The result will be displayed in the calculators display

The resulting number can be used as the first number in the next calculation

Numbers are rounded to a whole number

Numbers can be entered by clicking, or through the numbered keys
